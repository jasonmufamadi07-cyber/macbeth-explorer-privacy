Data Subject Rights – Macbeth Explorer

Last updated: 08 January 2026

As a user of Macbeth Explorer, you have the following rights regarding your personal data:

Access your data – You can request a copy of the personal information we hold about you (email and account information).

Correction – You can request corrections if any information is inaccurate or incomplete.

Deletion – You can request that your account and all associated data be permanently deleted.

Restriction of processing – You can request that we limit how we use your personal data.

Data portability – You can request a copy of your personal data in a structured, machine-readable format.

Withdraw consent – You can withdraw consent to the collection of your data at any time.
